## Feature additions

- [x] omittable clef and or timeSignature ([PR](https://github.com/markacola/react-vexflow/pull/4))
- [x] added accidentals using tonal
- [x] chords
- [x] beams (currently only quarter groups)
- [x] ties
- [ ] ties across staff break
- [x] dots
- [ ] colors
- [ ] chord symbols
- [ ] tuplets
- [ ] generate staves from rhythmical format
- [ ] jump signs
- [ ] multiple staves
- [ ] checkout https://github.com/0xfe/vexflow/wiki/Microtonal-Support !
- [ ] change clef/time signature mid score
- [my fork of react-vexflow](https://github.com/felixroos/react-vexflow)
