### typescript to doc / form

`npx typescript-json-schema tsconfig.json Person --include Person.ts`

- json schema could be used to display props in a doc
- can also add comments + TJS types
- could also generate forms for json schemas using: https://www.npmjs.com/package/json-schema-to-yup

### Ebooks

0. Make sure [DeDRM plugin](https://github.com/apprenticeharper/DeDRM_tools/releases) is installed in calibre
1. open ascm file with adobe digital editions => decrypts ascm file
2. Go to "~/Documents/Digital Editions" => there is the decrypted, but still scrambled file
3. Drag epub/pdf file to calibre => unscrambles file
